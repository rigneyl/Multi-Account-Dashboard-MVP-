# 📱 Multi-Account Dashboard (MVP)

A lightweight, privacy-first mobile-style web app that lets you **manually track balances** from multiple financial platforms—Wise, Revolut, PayPal, banks, crypto wallets, trading accounts, and more—all in a single, clean dashboard.

No API integrations. No logins. No cloud.
**Just a simple unified “Now View” of your money.**

---

## 🚀 Features

### ✔️ Core Features

* Add accounts manually (name, provider, currency, balance)
* Track balances for:

  * Wise
  * Revolut
  * PayPal
  * Bank accounts
  * Crypto wallets
  * Trading apps
* Automatic conversion to base currency (using demo FX rates)
* Total net balance display
* Edit, update, delete accounts
* Filter by All / Fiat / Crypto
* Light & Dark modes
* Fully mobile-friendly design

### ✔️ Tech Details

* Single-file HTML (no dependencies)
* Pure HTML/CSS/JS
* In-memory demo data
* Clean UI inspired by native mobile apps
* Ready for conversion to PWA or localStorage

---

## 🧩 Purpose

This app gives users who have money spread across multiple platforms a **quick, private, manual way** to see all their balances in one place.
Instead of connecting APIs or sharing sensitive data, users just update numbers themselves and get an instant consolidated view.
